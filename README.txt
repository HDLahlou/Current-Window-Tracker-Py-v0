This is a min viable product, a part of a one day challenge to explore python and its rudimentary ability to monitor user window access, on windows devices.

CURRENT VERSION: V0
CURRENT FUNCTIONALITY: 
- Recognizes windows of user
- Parses window to see if it contains a substring within "watchList"
- Can add or remove options from "watchList"
- Has basic buttons and input fields

FUTRE FUNCTIONALITY:
- Add time window conditionality to only function during certain time periods
- Record data into a database
